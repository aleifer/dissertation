*** An introduction ***

The HarvardThesis package provides all of files needed to support the production and typesetting of a PhD thesis at Harvard. The package strives to fully separate content from form in a way that LaTeX strives, but ultimately fails, to accomplish. Guided by the principle that nothing should exist in duplicate, the package includes a number of scripts that will build the final product (a beautiful pdf) and automatically remove all of the helper files.
